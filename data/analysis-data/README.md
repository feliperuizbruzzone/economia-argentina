# Analysis Data

Datos listos para analisis.

El archivo canonico actual del flujo AFIP es:

```text
20260710_afip_ganancias_sociedades_tidy_homologado.csv
```

Es un panel analitico largo sin comprimir, homologado por rama comun amplia y
versionable en GitHub porque queda por debajo de 100 MB. Sus columnas ya
incluyen anio fiscal, rama original, rama homologada, variable economica y valor
en pesos corrientes. Los diccionarios regenerables quedan en
`data/intermediate-data/afip-estadisticas-tributarias/` para trazabilidad.
