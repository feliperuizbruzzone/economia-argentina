# Documentation

Guardar aqui notas metodologicas, diccionarios de datos, bitacoras de cambios y
referencias de fuentes.

## Criterio De Organizacion

La documentacion especifica de una fuente se guarda en una subcarpeta propia:

```text
documentation/<fuente>/
```

En cada subcarpeta se elimina el prefijo redundante del nombre de archivo. Por
ejemplo, los antiguos `afip_*.md` viven ahora en `documentation/afip/`.

Fuentes actuales:

- `afip/`: AFIP/ARCA, anuario de estadisticas tributarias.
