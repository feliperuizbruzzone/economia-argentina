# Command Files

Todo script ejecutable del proyecto debe vivir bajo esta carpeta.

- `config/`: constantes y utilidades compartidas.
- `processing-command-files/`: descarga, extraccion, limpieza, validacion y armado de datos.
- `analysis-command-files/`: scripts de analisis, tablas, figuras y reportes.

Los scripts deben poder ejecutarse desde la raiz del repositorio.
